# PURPOSE

The CES Design System defines the visual identity of the Chetan Suraswal brand.

Every website, video, thumbnail, presentation, portfolio, social media post, logo, product and future digital experience must follow this design system.

Consistency builds recognition.

Every design decision should strengthen the CES identity.

---

# GOLDEN RULES

Every design decision must satisfy these questions.

1. Does it look premium?

2. Does it improve usability?

3. Does it strengthen the CES brand?

4. Is it timeless?

5. Would Apple simplify it further?

If the answer is "No" to any question,

reconsider the design.
